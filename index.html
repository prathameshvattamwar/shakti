Project Overview

Project Name: Shakti Pith Chhatrapati Shivaji Maharaj Construction

Client: Netaji Subhashchandra Bose Pratishthan

Location: Phaltan, Maharashtra, India

Budget: Rs. 13 Crore 87 Lakh

Objective: To construct a temple for Chhatrapati Shivaji Maharaj

Website Sections

  • Home: Introduction to the project, images, mission statement
  • About Us: Information about the trust, history, trustee list
  • Project Details: Detailed report, objectives, vision, scope, cost breakdown
  • Services: Proposed and improved services, disaster management plan
  • Museum & Statue: Museum and statue details, development plans
  • Infrastructure Development: Infrastructure and amenities development details
  • Donations: Contribution information, donation form, payment gateway
  • Contact Us: Contact form, address, location map, trustee contact details
  • Gallery: Photo gallery of the site, historical references, project progress

Web Design Elements

Color Scheme: Shades of blue and white to symbolize trust and purity.

Typography: Poppins font for a modern and clean look.

Images: High-quality images of Chhatrapati Shivaji Maharaj, site plans, and historical artifacts.

Layout: Clean, modern layout with a fixed navigation bar, responsive design for mobile compatibility.

Background: Subtle textures or gradients that don't distract from the content.

Backend Development

Framework: Use Django or Flask for backend development.

Database: PostgreSQL or MySQL for handling data.

Features:

  • User authentication and roles (admin, user, donor)
  • Content management system for updating project details
  • Donation management system
  • API for mobile application (if needed)

Frontend Technologies

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Bootstrap 5 for responsive design
  • React.js for interactive components (optional)

Suggested Background Web Design Images

Images of Chhatrapati Shivaji Maharaj, historical artifacts, ancient Shakti Pith temples, concept art or architectural renderings
